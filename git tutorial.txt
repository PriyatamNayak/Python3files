			pushing a new file on github
echo "#python">> README.md for python files or repo				
1. 	remove any .git folder in ur files
2. 	git init initialize the folder with git
3. 	ls will show all ur files
4.	git config --global user.email "amitbora007@gmail.com"
	git config --global user "Amit Bora"
5. 	git add .  adds the file to local repository
6. 	git status shows the status of ur files any new file will have new written before it
7. 	git commit -m 'first commit'
8. 	git remote add origin git@github.com:amitbora007/learn.git or  
	git remote add origin https://github.com/amitbora007/python.git

9. 	git push -u origin master
